软件介绍：
        本程序用Python编写，并在Python平台上运行，实现了增量式sfm稀疏重建方法。

使用步骤：
        ① 材料准备：
	使用者需单独创建一文件夹，文件夹中放入需要重建的图片以及写入内参矩阵K的txt文件。
	图片命名最好按照实际拍摄时的顺序命名为"xxx001", "xxx002"如此，以保证程序重建顺利。
	相机内参矩阵需要事先标定好，并且写入"K.txt"，写入格式为3行，如：
		[3293.7,      0, 2052.2]
		[0,      3296.9, 1545.9]
		[0,           0,      1]
        ② 程序运行：
	找到名为3D_reconstruction.py的文件并运行，即可开始三维重建

        ③ 参数调整：
	在config.py中，列了一些对三维重建有影响的参数，使用者可以根据自己的需求更改其中变量的数值。
